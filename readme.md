# 借助Window performance-监控网页程序性能
- 1. 了解performance的各个属性代表的意思
- 使用 performance.timing 信息简单计算出网页性能数据
- 使用 performance.getEntries() 获取所有资源请求的时间数据
- 使用 performance.now() 精确计算程序执行时间
- 使用 performance.mark() 也可以精确计算程序执行时间
